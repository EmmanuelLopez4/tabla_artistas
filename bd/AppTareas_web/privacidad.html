Política de Privacidad

Inicio · ISO 27000

Resumen

En este Gestor de Tareas tratamos datos personales mínimos necesarios para la operación del servicio: nombre, correo y contenido de las tareas. No compartimos datos con terceros sin consentimiento explícito salvo requerimiento legal.

Finalidad

  • Gestión y asignación de tareas.
  • Notificaciones y recordatorios.
  • Auditoría y control interno para seguridad de la información.

Conservación

Los datos se conservan mientras el usuario use el servicio. En entornos productivos, los backups se conservarían según políticas internas y leyes aplicables.

Derechos

Los usuarios pueden solicitar acceso, rectificación o supresión de sus datos mediante contacto a: seguridad@tu-dominio.com
